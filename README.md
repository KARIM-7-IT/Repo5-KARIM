# The Karbala Auto Tech Web Deployment Project

## 🛠️ Project Overview
This repository contains the deployment files for a high-fidelity, responsive 3-page web platform engineered for Karbala Auto Tech—a specialized center for vehicle digital diagnostics, ECU adaptations, and electrical system troubleshooting.

The project features a modern dark-theme system, native Arabic language alignment (RTL), custom typography via Google Fonts (Cairo), and fully interconnected multi-page navigation.
